#### 2、自定义协议数据传输实现

```
Utils/sortInterface.go文件
 实现了自定义数据传输协议，可以根据不同类型数据传输，并进行数据排序处理；
 在服务器收到客户端请求，对数据做对应的处理并按照请求协议返回数据功能实现
	
协议：
 数据传输开始定义：
  0 0 0 和 1 0 0 
   其中第一个数如果为0表示使用内存处理数据，为1表示使用文件形式处理数据
 数据文件传输：
  1 1 0 0
   首位：如果为0表示使用内存处理数据，为1表示使用文件形式处理数据
   第2位：1表示整数，2表示浮点数，3表示字符串，4表示结构体
   第3位：如果是整数和浮点数第3位为要传输的数字，如果是字符串和结构体第三位为其对应的字节长度
   其余为字符串和结构体字节文件
 数据传输结束定义：
  0 0 1 和 0 0 2
   首位：如果为0表示使用内存处理数据，为1表示使用文件形式处理数据
   第2位：固定为0
   第3位：1表示生序排列，2表示降序排列
```
